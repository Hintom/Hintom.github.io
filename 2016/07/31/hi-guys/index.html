windows Hexo+Github 建立博客

废了半天劲终于搞定了，下面是一些攻略

基本步骤

安装 github,nodejs,hexo,按照这篇博客来

一些问题

安装过程中我遇到一些问题，
一：首先安装 github windows desktop 结果下载到百分之三十几停住了 解决方案：下载离线包。

二：用 github–powershell时识别不了 npm 命令 在里面设置下环境变量就ok：
bash $Env:Path = $Env:Path +"<npm的路径>"bash
然后装hexo，再添加hexo的路径。但是重启后用hexo提示找不到node，google发现是环境变量的问题，用
bash $env:path -split ';' | SelectString-nodejsbash
查找nodejs的路径，原来把一个;打成了：，此时要修改环境变量怎么办，解决方法，先把path导出
bash $env:path<<a.outbash
然后修改a.out再复制出来。

hexo的一些命令

1.新建文章
bash hexo new "text" 新建文章bash

2.生成
bash hexo generatebash

3.部署到git
bash hexo deploybash

合并起来简写就是
bash hexo d -gbash

hexo的编辑

text 点击tetx可以链接到url
“##” 二级标题
“###” 三级标题
