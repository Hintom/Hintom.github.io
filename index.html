ruby

attr_accessor

那么 name变量既可读也可写
class Person
attr_accessor :name

def greeting
“Hello #{@name}”
end
end

person = Person.new
person.name = “Dennis”
person.greeting # => “Hello Dennis”

1…10 创建从 1 到 9 的范围
